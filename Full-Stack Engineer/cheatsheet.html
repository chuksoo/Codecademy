Common CSS Properties

Cheat Sheet

Box Properties

Property Description Value(s)
background-color: Sets the background color of an element.
  • Keyword - ex. red
  • Hex - #bbff00, #bf0, #bbff00ff, #bfof
  • RGB/RGBA - rgb(255, 255, 128), rgba(117, 190, 218, 0.75)
  • HSL/HSLA - hsl(50, 33%, 25%), hsla(50, 33%, 25%, 0.75)
  • Special - currentcolor, transparent
background-image: Sets one or more background images on an element.
  • Keyword - none
  • Image - url(), <gradient&rt, element(), image(), cross-fade(), image-set()
border: Shorthand CSS property sets an element's border. It sets the values of border-width, border-style, and border-color. Order does not matter.
  • Style - solid, dotted, dashed, etc.
  • Width - 2px
  • Color - Keyword, Hex, RGB, etc.
display:

The display CSS property sets whether an element is treated as a block or inline box and the layout used for its children, such as flow layout, grid or flex.

Formally, the display property sets an element's inner and outer display types. The outer type sets an element's participation in flow layout; the inner type sets the layout of children. Some values of display are fully defined in their own individual specifications; for example the detail of what happens when display: flex is declared is defined in the CSS Flexible Box Model specification.
  • Precomposed - block, flex, grid, flow-root, inline(-block, -flex, -grid)
  • Box Generation - none, contents
  • Multi-keyword - (block, inline) (flow, flow-root, flex, grid)
  • Other - table, table-row, list-item
float: Places an element on the left or right side of its container, allowing text and inline elements to wrap around it. The element is removed from the normal flow of the page, though still remaining a part of the flow (in contrast to absolute positioning).
  • Keyword - left, right, none, inline-start, inline-end
height: specifies the height of an element. By default, the property defines the height of the content area. If box-sizing is set to border-box, however, it instead determines the height of the border area.
  • Length - 120px, 10em, 100vh
  • Percentage - 75%
  • Keyword - max-content, min-content, fit-content(20em), auto
justify-self: Sets the way a box is justified inside its alignment container along the appropriate axis.
  • Basic - auto, normal, stretch
  • Positional - center, start, end, left, right, etc.
  • Baseline - baseline, first baseline, last baseline
  • Overflow - safe center, unsafe center
margin: Shorthand property sets the margin area on all four sides of an element in a clockwise direction from the Top.
  • 4-sides - margin: 1em; margin: -3px;
  • T/B+L/R - margin: 5% auto;
  • T+L/R+B - margin: 1em auto 2em;
  • T+B+L+R - margin: 2px 1em 0 auto;
max/min-height/width: Sets the maximum height/width of an element. It prevents the used value of the height/width property from becoming larger/smaller than the value specified for max/min-height/width.
  • Length - 3.5em, 10px
  • Percentage - 75%
  • Keyword - none, max-content, min-content, fit-content(20em)
opacity: Sets the opacity of an element. Opacity is the degree to which content behind an element is hidden, and is the opposite of transparency.
  • Alpha-value - number between 0 and 1
  • Percentage - 75%
overflow: Shorthand property sets the desired behavior when content does not fit in the parent element box (overflows) in the horizontal and/or vertical direction. Shorthand for overflow-x: and overflow-y:
  • Keyword - visible, hidden, clip, scroll, auto, hidden visible
padding: Shorthand property sets the padding area on all four sides of an element in a clockwise direction from the Top.
  • 4-sides - margin: 1em; margin: -3px;
  • T/B+L/R - margin: 5% auto;
  • T+L/R+B - margin: 1em auto 2em;
  • T+B+L+R - margin: 2px 1em 0 auto;
position: Sets how an element is positioned in a document. The top, right, bottom, and left properties determine the final location of positioned elements.
  • Keyword - static, relative, absolute, fixed, sticky
  • Properties - top: 40px, bottom: 40px, left: 40px, right:40px
visibility: Shows or hides an element without changing the layout of a document. The property can also hide rows or columns in a <table&rt.
  • Keyword - visible, hidden, collapse
width: Specifies the width of an element. By default, the property defines the width of the content area. If box-sizing is set to border-box, however, it instead determines the width of the border area.
  • Length - 120px, 10em, 100vh
  • Percentage - 75%
  • Keyword -
z-index: Sets the z-order of a positioned element and its descendants or flex items. Overlapping elements with a larger z-index cover those with a smaller one.
  • Keyword - auto
  • Integer - 0, 3, 289, -1 (negative values to lower the priority)

FlexBox and Grid

Property Description Value(s)
align-content: Sets the distribution of space between and around content items along a flexbox's cross-axis or a grid's block axis.
  • Positional - center, start, end, flex-start, flex-end
  • Normal - normal
  • Baseline - baseline, first baseline, last baseline
  • Distributed - space-between, space-around, space-evenly, stretch
  • Overflow - safe center, unsafe center
align-items Sets the align-self value on all direct children as a group. In Flexbox, it controls the alignment of items on the Cross Axis. In Grid Layout, it controls the alignment of items on the Block Axis within their grid area.
  • Basic - normal, stretch
  • Positional - center, start, end, flex-start, flex-end, self-start, self-end
  • Baseline - baseline, first baseline, last baseline, safe center, unsafe center
align-self Overrides a grid or flex item's align-items value. In Grid, it aligns the item inside the grid area. In Flexbox, it aligns the item on the cross axis.
  • Keyword - auto, normal
  • Positional - center, start, end, flex-start, flex-end, self-start, self-end
  • Baseline - baseline, first baseline, last baseline, stretch
  • Overflow - safe center, unsafe center
align-tracks Sets the alignment in the masonry axis for grid containers that have masonry in their block axis.
  • Keyword - start, space-between, center, (start, center, end)
flex: Shorthand property sets how a flex item will grow or shrink to fit the space available in its flex container.
  • Keyword - auto, initial, none
  • flex-grow - flex: 2
  • flex-basis - flex: 10px, flex: 30%, flex: min-content
  • flex-grow/basis - flex: 1 20px
  • flex-grow/shrink - flex: 1 2
  • flex-grow/shrink/basis - flex: 2 1 30px
flex-basis Sets the initial main size of a flex item. It sets the size of the content box unless otherwise set with box-sizing.
  • Width - 10em, 3px, 50%, auto
  • Intrinsic sizing - max-content, min-content, fit-content
  • Automatic - content
flex-direction Sets how flex items are placed in the flex container defining the main axis and the direction (normal or reversed).
  • normal - row, column
  • reversed - row-reverse, column-reverse
flex-flow: Shorthand property specifies the direction of a flex container, as well as its wrapping behavior.
  • flex-direction - row, roe-reverse, column, column-reverse
  • flex-wrap - nowrap, wrap, warp-reverse
  • flex-direction/wrap - row nowrap, column wrap, column-reverse wrap-reverse
flex-grow: Sets the flex grow factor, which specifies how much of the flex container's remaining space should be assigned to the flex item's main size. When the flex-container's main size is larger than the combined main size's of the flex items, the extra space is distributed among the flex items, with each item growth being their growth factor value as a proportion of the sum total of all the container's items' flex grow factors.
  • Number values - 3, 0.6
flex-shrink: Sets the flex shrink factor of a flex item. If the size of all flex items is larger than the flex container, items shrink to fit according to flex-shrink. In use, flex-shrink is used alongside the other flex properties flex-grow and flex-basis, and normally defined using the flex shorthand.
  • Number values - 3, 0.6
flex-wrap; Sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.
  • Keyword - wrap, nowrap, wrap-reverse
justify-content: Defines how the browser distributes space between and around content items along the main-axis of a flex container, and the inline axis of a grid container.
  • Positional - center, start, end, flex-start, flex-end, left, right
  • Normal - normal
  • Distributed - space-between, space-around, space-evenly, stretch
  • Overflow - safe center, unsafe center
justify-tracks: Sets the alignment in the masonry axis for grid containers that have masonry in their inline axis.
  • Keyword - start, space-between, center, (start, center, end)
grid: Shorthand property that sets all of the explicit and implicit grid properties in a single declaration.
  • grid-template
  • grid-template-rows
  • grid-template-columns
  • Global
grid-area: Shorthand property specifies a grid item's size and location within a grid by contributing a line, a span, or nothing (automatic) to its grid placement, thereby specifying the edges of its grid area.
  • Keyword: auto...auto / auto / auto / auto
  • Custom-ident values: some-grid-area, some-grid-area / another-grid-area
  • Span and Integer values: 2 / span 3
  • Global

Text Properties

Property Description Value(s)
color: Sets the foreground color value of an element's text and text decorations, and sets the currentcolor value. currentcolor may be used as an indirect value on other properties and is the default for other color properties, such as border-color.
  • Keyword - ex. red
  • Hex - #bbff00, #bf0, #bbff00ff, #bfof
  • RGB/RGBA - rgb(255, 255, 128), rgba(117, 190, 218, 0.75)
  • HSL/HSLA - hsl(50, 33%, 25%), hsla(50, 33%, 25%, 0.75)
  • Special - currentcolor, transparent
font-family: Specifies a prioritized list of one or more font family names and/or generic family names for the selected element.
  • Family-name - Times, Helvetica 'Comic Sans MS'
  • Generic-name - serif, sans-serif, monospace, cursive, etc.
font-size: Sets the size of the font. Changing the font size also updates the sizes of the font size-relative <length&rt units, such as em, ex, and so forth.
  • Absolute-size - xx-small, small, medium, large, xxx-large
  • Relative-size - smaller, larger
  • Length - 12px, 0.8em
  • Percentage - 80%
  • Math - math
font-style: Sets whether a font should be styled with a normal, italic, or oblique face from its font-family.
  • Keyword - normal, italic, oblique, oblique 10deg
font-weight: Sets the weight (or boldness) of the font. The weights available depend on the font-family that is currently set.
  • Keyword - normal, bold
  • Relative-keyword - lighter, bolder
  • Numeric - number from 1 to 1000
line-height: Sets the height of a line box. It's commonly used to set the distance between lines of text. On block-level elements, it specifies the minimum height of line boxes within the element. On non-replaced inline elements, it specifies the height that is used to calculate line box height.
  • Keyword - normal
  • Unitless - 3.5
  • Length - 3em
  • Percentage - 34%
list-style: Shorthand property allows you to set all the list style properties (image, position, and type) at once
  • Type - disc, circle, square, "-", etc
  • Image - none, url("../img/shape.png") <gradient&rt
  • Position - inside, outside
  • Keyword - none
text-align: Sets the horizontal alignment of the inline-level content inside a block element or table-cell box. This means it works like vertical-align but in the horizontal direction.
  • Keyword - start, end, left, right, center, justify, justify-all, match-parent
  • Character-based - ".", "." center
  • Block alignment - -moz-center, -webkit-center
text-transform: Specifies how to capitalize an element's text. It can be used to make text appear in all-uppercase or all-lowercase, or with each word capitalized. It also can help improve legibility for ruby.
  • Keyword - none, capitalize, uppercase, lowercase, full-width, full-size-kana
verticlal-align: Sets vertical alignment of an inline, inline-block or table-cell box.
  • Keyword - baseline, sub, super, text-top, text-bottom, middle, top, bottom
  • Length - 10em, 4px
  • Percentage - 20%
word-spacing: Sets the length of space between words and between tags.
  • Keyword - normal
  • Length - 3px, 0.3em
